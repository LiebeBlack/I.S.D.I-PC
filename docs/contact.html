Preguntas Frecuentes

Resolvemos tus Dudas

Respuestas a las preguntas más comunes sobre E.E.D.A.

¿Es E.E.D.A. completamente gratuito?

Sí, E.E.D.A. es un proyecto open source y completamente gratuito. Puedes usarlo, modificarlo y distribuirlo libremente bajo los términos de la licencia MIT.

¿Qué edad recomendada tienen los módulos?

E.E.D.A. está diseñado para jóvenes de 3 a 16 años, con contenido adaptado a cada grupo etario mediante nuestros protocolos CPA (Concreto-Pictórico-Abstracto).

¿Necesito conocimientos técnicos para instalarlo?

No, hemos creado guías paso a paso muy detalladas. Solo necesitas tener Python 3.12+ instalado y seguir los comandos de instalación.

¿Puedo contribuir al proyecto?

¡Absolutamente! Aceptamos contribuciones de todo tipo: código, documentación, traducciones, reporte de errores y sugerencias. Visita nuestro GitHub para más información.
